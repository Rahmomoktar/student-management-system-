## Project Description
This is a Student Management System (SMS) project built using HTML, CSS, and JavaScript.
It demonstrates Git workflow including branching, merging, and conflict resolution.

## Git Commands Used

- git init → Initializes a new Git repository
- git add . → Stages all files for commit
- git commit -m "message" → Saves changes in history
- git branch → Shows or creates branches
- git merge → Combines branches
- git push → Sends code to GitHub
- git pull → Fetches updates from GitHub
